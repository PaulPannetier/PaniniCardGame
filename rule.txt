Règles du jeu :

- Un joueur peut poser des cartes que sur son côté du terrain.

Au début du tour du joueur :

- Les effets de début de tour s'active.
- Le joueur pioche une carte.

Pendant le tour du joueur : 

- Peut joué un joueur de son coté du terrain.
- Peut joué autant de sort que voulu.
- Chaque joueur peut effectué une action :
	- se déplacé sur une case adjacente (conservation de la possétion de balle).
	- effectué un duel contre un ennemi sur un case adjacente si l'un des 2 possède la balle.


pendant le tour de l'adversaire : 
le joueur ne peut rien faire.

Duel :  

- une carte défend, une ou plusieurs attaques :
	Point attaque(PA) = Somme attaque des carte attaquantes
	Point de defence(PD) = defence carte qui défend
	=> PA > PD => attaquant gagne et le defenseur exclue du terrain, le premier attaquant passe sur ca case avec le ballon, les autres ne bouge pas.
	=> PA <= PD => défenceur gagne, tout les attaquant sont virer du terrain.
